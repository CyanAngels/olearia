What is Olearia


Olearia is DAISY digital talking book playback software for Apple Macintosh. Olearia version 0.90b will play back audio only and full text full audio DAISY digital talking books in both DAISY 2.02 and DAISY/NISO format. Future released versions of Olearia will also support DAISY text only format and text display.


This version of Olearia is still beta and several features are not yet implemented. In particular these include:


  • Navigation by page number
  • Bookmarking
  • Reading of Text Only DAISY books


These features will be added to the final released version of Olearia.
