Opening a book



To open a book in Olearia you may use the Open item under the File menu or select command O. You can then select either the folder containing the book files or the control files themselves. Only allowed files will be available for selection.



  • .opf

  • .ncx

  • ncc.html



The first two types are found in DAISY/NISO 2005 books. The ncc.html file is found in DAISY 2.02 books. It is also possible to open DAISY/NISO books by double clicking on the opf or ncx files in the finder.



Olearia keeps a list of recent books in the Recent Books submenu of the File menu, choosing one of these will open the book at its most recent playback position.

Playback



You can start and stop playback of a book by pressing the spacebar on you keyboard, selecting play or pause in the Navigation menu or by pressing the on screen Play/Pause button.
