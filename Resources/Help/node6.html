Navigation


Daisy offerers several advantages over other audio book formats. One of the major ones is navigation. Navigation is set on a per book basis and permits you to move to a chapter, Navigation is by levels. You move up and down by levels using the up and down arrow keys on the keyboard or by using the on screen navigation buttons.


You can move from one navigation level to the next by using the right and left arrow keys or by using the next and on screen buttons.


You can also move ahead and back within a given level by using the Fast forward and Fast Back on screen buttons or Command-right arrow for forward and Command-left arrow for back. The amount that you move forward and back is controls in the program preferences. Default is 30 seconds.
