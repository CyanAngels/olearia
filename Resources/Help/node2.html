What is Olearia?



Olearia is DAISY digital talking book playback software for Mac OS X. Olearia will play back audio only and full text full audio DAISY digital talking books in both DAISY 2.02 and DAISY/NISO V3 format. Future updates of Olearia will also support DAISY text only format and text display.



This version of Olearia is still beta and several features are not yet implemented. In particular these include:



  • Navigation by page number

  • Bookmarking

  • Reading of Text Only DAISY books
